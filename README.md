# ANGULAR
Projeto para o estudo de Angular na disciplina DW2A4.

<br>

## Projetos

### 1. EXAMPLE01
Aplicação funcional que solicita requisições na [API criada em Java com Spring Boot](https://github.com/leonarita/ProjetoES4A4).

### 2. EXAMPLE02
Aplicação simplista para exemplificar a estruturação de uma aplicação com microfrontends.

### 3. EXAMPLE03
Aplicação simplista para exemplificar o uso das estruturas Angular.

<br>

## Pré-Requisitos

1. Instalar NodeJS : https://nodejs.org/en/
2. Instalar Angular: Comando 'npm install –g @angular' no terminal após instalar o NodeJS
3. Instalar o git: https://git-scm.com/

<br>

## Acessar projeto Angular

1. Dar clone na máquina local: Comando 'git clone https://github.com/leonarita/ProjetoDW2A4.git' no git
2. Baixar as dependências: Comando 'npm i' no terminal nas pastas 'EXAMPLE01', 'EXAMPLE02' e 'EXAMPLE03' 
3. Rodar o projeto: Comando 'ng serve' no terminal na pasta do projeto desejado
4. Acessar http://localhost:4200
